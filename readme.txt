=== Off-Canvas Sidebars ===
Contributors: keraweb
Tags: genesis, off-canvas, sidebars, slidebars, jQuery, app, mobile, tablet, responsive
Requires at least: 3.8
Tested up to: 4.4
Stable tag: 0.1

Add off-canvas sidebars using the Slidebars jQuery plugin.
This plugin should work for most themes though I can't be sure it will work on all themes. If you find a problem, please let me know.

== Description ==

= Overview / Features =

*	Add off-canvas sidebars to the left and right of your website
*	You can add the control buttons with a widget, a menu item or with custom code, click here for documentation.
*	Various customisation options inder your Appearances menu

= Compatibility =

I think this plugin will work with most other plugins allthough it could be possible that heavy customized themes are not compatible. Please let me know if you have issues!

= It's not working! / I found a bug! =

Please let me know through the support and add a plugins and themes list! :)

== Installation ==

Installation of this plugin works like any other plugin out there. Either:

1. Upload the zip file to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Or search for Off-Canvas Sidebars" via your plugins menu.
